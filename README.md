# Gradient Boosting Classifier - Poster Project

This repository contains the notebook used for the poster presentation on Gradient Boosting in Predictive Analytics.

## Files
- `gradient_boosting.ipynb`: Main notebook with preprocessing, model training, evaluation, and references.

## How to Run
1. Open the notebook in Google Colab or Jupyter.
2. Install dependencies if needed (`scikit-learn`, `pandas`, `matplotlib`).
3. Run all cells.

## Dataset
Breast Cancer Wisconsin dataset (available in scikit-learn).
