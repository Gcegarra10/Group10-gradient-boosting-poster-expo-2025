
# Gradient Boosting Classifier - Breast Cancer Dataset

## Purpose
This notebook demonstrates the use of a Gradient Boosting Classifier to predict breast cancer types using the Breast Cancer Wisconsin dataset. 
The notebook includes data preprocessing, model training, evaluation, and visualization.

## Dataset
- Name: Breast Cancer Wisconsin (Diagnostic) Dataset
- Source: [scikit-learn datasets](https://scikit-learn.org/stable/modules/generated/sklearn.datasets.load_breast_cancer.html)
- License: Public domain / scikit-learn dataset (check https://scikit-learn.org/stable/datasets/index.html)
- Features: Numeric features extracted from breast cancer images
- Target: 0 = malignant, 1 = benign

## Execution Steps
1. Install required packages from `requirements.txt`.
2. Open `gradient_boosting.ipynb` in Jupyter or Google Colab.
3. Run all cells sequentially.
4. Observe metrics, visualizations, and interpretation of results.
